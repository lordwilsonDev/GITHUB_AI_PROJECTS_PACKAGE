## üí° BEST PRACTICES

### 1. Recording Experiences

**DO:**
```python
memory.record_breakthrough({
    'title': 'Clear, specific title',
    'description': 'Detailed explanation',
    'domain': 'specific_field',
    'vdr_score': 0.95,  # Validation confidence
    'synthesis_from': [node_ids]  # What led to this
})
```

**DON'T:**
```python
memory.record_experience({'stuff': 'happened'})  # Too vague
```

### 2. Querying Efficiently

**DO:**
```python
# Specific queries
results = memory.semantic_search('consciousness multiplication', limit=10)

# Type-filtered queries
breakthroughs = memory.query_by_type('Breakthrough')

# Time-bounded queries
recent = memory.query_by_time_range(last_week, now)
```

**DON'T:**
```python
# Get ALL nodes at once (expensive!)
all_nodes = list(memory.nodes.values())  # Slow for large graphs
```

### 3. Creating Connections

**DO:**
```python
# Meaningful edge types
memory.add_edge(source, target, 'SYNTHESIZED_INTO')

# Include metadata
memory.add_edge(source, target, 'INFLUENCED', 
                metadata={'strength': 0.8, 'mechanism': 'validation'})
```

**DON'T:**
```python
# Generic edges
memory.add_edge(source, target, 'RELATED_TO')  # Not specific enough
```

### 4. Handling Contradictions

**DO:**
```python
# Mark contradictions explicitly
memory.add_edge(idea1, idea2, 'CONTRADICTS',
                metadata={'paradox_type': 'wave_particle_duality'})

# Record as breakthrough opportunity
memory.record_insight({
    'type': 'productive_contradiction',
    'nodes': [idea1, idea2],
    'resolution_strategy': 'higher_order_synthesis'
})
```

**DON'T:**
```python
# Ignore or delete contradictions - they drive breakthroughs!
```
