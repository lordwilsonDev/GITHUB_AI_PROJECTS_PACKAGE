AI ORCHESTRATION SYSTEM - START HERE
========================================

Your 620-agent AI orchestration system is ready to deploy!

QUICK START (Copy and paste these commands into Terminal):

1. Open Terminal:
   Press Cmd+Space, type "Terminal", press Enter

2. Navigate to project:
   cd ~/ai-agent-orchestration

3. Make scripts executable:
   chmod +x *.sh

4. Deploy the system:
   ./EXECUTE_DEPLOYMENT.sh

   OR use the simple start script:
   ./start.sh

That's it! The script will:
- Check Docker is running
- Start all 8 services
- Display dashboard URLs
- Show system status

========================================
TROUBLESHOOTING
========================================

If Docker is not running:
  open -a Docker
  sleep 30
  ./EXECUTE_DEPLOYMENT.sh

To check status:
  ./status.sh

To view logs:
  docker-compose logs -f

To stop system:
  ./stop.sh

To reset everything:
  docker-compose down -v
  ./EXECUTE_DEPLOYMENT.sh
