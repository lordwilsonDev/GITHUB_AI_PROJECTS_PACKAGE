## üí° PERFORMANCE TIPS

**First run is SLOW:**
- Each model loads separately
- Ollama initializes fresh contexts
- ~10-15 minutes total

**Subsequent runs FASTER:**
- Models stay in memory
- Context reuse
- ~3-5 minutes total

**Speed optimization:**
- Use smaller models (phi3, gemma2:2b)
- Reduce timeout values
- Run on machines with more RAM
