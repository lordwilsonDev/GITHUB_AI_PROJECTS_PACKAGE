## üöÄ Quick Start

### Prerequisites

- Python 3.8+
- macOS (M1 optimized)
- Docker (optional, for containerized deployment)
- 16GB+ RAM recommended

### Installation

```bash
# Clone the repository
git clone https://github.com/lordwilsonDev/vy-cognitive-sovereignty-stack.git
cd vy-cognitive-sovereignty-stack

# Install master dependencies
pip install -r requirements_master.txt

# Initialize each system (see individual system READMEs)
cd love-engine-real && pip install -r requirements.txt
cd ../sovereign-stack && pip install -r requirements.txt
# ... repeat for other systems
```

### First Run

```bash
# Start Love Engine (Safety First!)
cd love-engine-real
python main.py

# Start Sovereign Stack
cd ../sovereign-stack
./ignition.sh

# Start Vy-Nexus Infrastructure
cd ../vy-nexus
python nexus_core.py
```
