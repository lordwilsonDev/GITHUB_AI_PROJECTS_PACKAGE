## How The Witness Works

### 1. Observation Phase

The Witness reads:
- **Logs** - All system activity (Motia, Love Engine, RAY, metrics)
- **Metrics** - Performance data, health checks, task completions
- **TODOs** - Task progress, blockers, completion rates

It looks back 24 hours by default (configurable).

### 2. Pattern Extraction

The Witness identifies:
- **Error patterns** - What's failing and why
- **Success patterns** - What's working well
- **Love vetoes** - When the Heart blocks unsafe actions
- **Task patterns** - Completion rates, blockers
- **Health patterns** - System resource usage
- **Emergent behaviors** - Unexpected patterns

### 3. Insight Generation

For each pattern, The Witness generates:
- **Observation** - What it saw
- **Severity** - How important it is
- **Recommendation** - What to do about it

### 4. Wisdom Accumulation

Insights are stored in two formats:
- **insights.jsonl** - Machine-readable log (for future AI analysis)
- **README.md** - Human-readable summary (for you)

Wisdom grows over time. The system learns from its history.
